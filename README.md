# Spayce

**Nigeria's workspace marketplace.** Find and book desks, meeting rooms, and private offices across Lagos, Abuja, and beyond.

## About

Spayce connects workspace seekers with space owners. Rent by the hour, day, week, or month. Instant booking, transparent pricing in Naira (₦).

## Tech Stack

- **Framework:** Next.js 15 (App Router)
- **UI:** React 19, Tailwind CSS, Radix UI
- **State/Data:** TanStack Query
- **Forms:** React Hook Form, Zod
- **Animations:** Framer Motion

## Getting Started

```bash
# Install dependencies
npm install

# Set environment variables (create .env.local)
# NEXT_PUBLIC_API_BASE_URL=your-api-url
# NEXT_PUBLIC_SITE_URL=https://spayce.ng

# Run development server
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) to view the app.

## Project Structure

- `app/` – Pages and layouts
- `components/` – Reusable UI components
- `lib/` – API client and utilities
